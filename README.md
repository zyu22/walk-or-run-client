## 📁 프로젝트 구조
```
Copysrc/
├── 📂 assets/         # 이미지, 폰트 등 정적 파일
├── 📂 components/     # 재사용 가능한 Vue 컴포넌트
├── 📂 router/         # Vue Router 설정
├── 📂 store/          # Vuex 상태 관리
├── 📂 views/          # 페이지 컴포넌트
└── 📂 services/       # API 통신 관련 서비스
```
